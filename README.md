# Scripts φαρμοσμένης Βελτιστοποίησης

## Γεωμετρική μέθοδος
Έγραψα το script `geometriki.py` το οποίο υπολογίζει το μέγιστο ή ελάχιστο συνάρτησης, αριθμητικά, για n διαστάσεις χρησιμοποιώντας την γεωμετρική μέθοδο (Ενότητα 6, 1ο μέρος) (μέχρι στιγμής δουλεύει καλά **μόνο** στις 2 αλλά ετοιμάζω και τις 3).

### Οδηγίες
- **ΑΠΑΙΤΟΥΝΤΑΙ οι βιβλιοθήκες `numpy` και `prettytable`** 
- Ορίστε σωστά την αντικειμενική συνάρτηση ($x_1$ --> `x[0]`, $x_2$ --> `x[1]`).
- Επιλέξτε αποδεκτό `a` (πλευρά τριγώνου/περιθώριο σφάλματος).
- Επιλέξτε το `goal = 'Min'` ή `'Max'`, ανάλογα με το τι ψάχνετε.
- Αν θέλετε η έρευνα να ξεκινήσει από κάποιο συγκεκριμένο τρίγωνο πρέπει να το ορίσετε εσείς (η μεταβλητή `start` περιέχει και τις 3 ακμές του 1ου τριγώνου έρευνας ειναι τύπου `ndarray` με `shape(3,2)`). Σε αντίθετη περίπτωση μπορείτε να χρησιμοποιήσετε την συνάρτηση `starting_shape_for_2_dimensions(a)`.
- Η τιμή της `rounding_decs` επηρεάζει **μόνο** την παρουσίαση των αποτελεσμάτων στο terminal (δηλαδή πόσα δεκαδικά ψηφία θα φαίνονται). Οι υπολογισμοί **δεν** επηρεάζονται.
