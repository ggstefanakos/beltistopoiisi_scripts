# Scripts φαρμοσμένης Βελτιστοποίησης

## Γεωμετρική μέθοδος
Έγραψα το script `geometriki.py` το οποίο υπολογίζει το μέγιστο ή ελάχιστο συνάρτησης για n διαστάσεις (μέχρι στιγμής δουλεύει καλά μόνο στις 2 αλλά ετοιμάζω και τις 3).

### Οδηγίες
- **ΑΠΑΙΤΟΥΝΤΑΙ οι βιβλιοθήκες `numpy` και `prettytable`** 
- Ορίστε σωστά την αντικειμενική συνάρτηση `f`
- Επιλέξτε το `goal = 'Min'` ή `goal = 'Max'`
- Επιλέξτε αποδεκτό `a` (πλευρά τριγώνου/περιθώριο σφάλματος)
- Αν θέλετε η έρευνα να ξεκινήσει από κάποιο συγκεκριμένο τρίγωνο πρέπει να το ορίσετε εσείς. Σε αντίθετη περίπτωση μπορείτε να χρησιμοποιήσετε την συνάρτηση `starting_shape_for_2_dimensions(a)`
- Η τιμή της `rounding_decs` επιρεάζει μόνο την παρουσίαση των αποτελεσμάτων στο terminal (δηλαδή πόσα δεκαδικά ψηφία θα φαίνονται). Οι υπολογισμοί δεν επιράζονται.

### Αποτελέσματα
Το ελάχιστο της συνάρτησης $f(x)=2x_1^2 + 10x_2^2 + 3\sin(x_1) + 8\cos(x_2) -10$ υπολογίζεται οπώς φαίνεται παρακάτω.
